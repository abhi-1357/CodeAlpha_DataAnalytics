## ✅ Tasks Completed

### 🕷️ Task 1 — Web Scraping
- **Target site:** [books.toscrape.com](http://books.toscrape.com) (100% legal practice site)
- **Data collected:** Title, Price, Rating, Availability across 1000 books (50 pages)
- **Libraries:** `requests`, `BeautifulSoup`, `pandas`
- **Output:** `scraped_books.csv` + 5 analysis charts

### 🔍 Task 2 — Exploratory Data Analysis
- **Dataset:** Titanic Survival Dataset (via Seaborn)
- **Analysis:** Missing value heatmap, survival by gender/class/age/fare, correlation matrix
- **Libraries:** `pandas`, `numpy`, `matplotlib`, `seaborn`
- **Key Finding:** Female passengers had 74% survival rate vs 19% for males

### 📊 Task 3 — Data Visualization
- **Dataset:** Netflix-style content dataset (1000 entries)
- **Visuals:** Dark-themed Netflix dashboard, genre heatmap, trend line charts, duration boxplots
- **Libraries:** `matplotlib`, `seaborn`, `pandas`, `numpy`
- **Output:** 3 publication-quality chart files
